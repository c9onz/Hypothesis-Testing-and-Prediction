# Hypothesis-Testing-and-Prediction
A repo demonstrating linear, quadratic, and exponential models to predict share price movement, hypothesis testing to predict expected salaries of college students, and variance among factors using Excel's Data Analytics tool. The last project uses Anova two-factor with replication to explore time differences of pain relief tablets that dissolve in water.
