# Hypothesis-Testing-and-Prediction
A repo demonstrating single and two factor ANOVA models with and without replication to observe effects and interaction of factors, and to test different hypotheses against various predictive models (Linear, Quadratic, Exponential, Regression) in R and Excel's Data Analytics Toolpak.
